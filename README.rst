===========
autocorrect
===========
Python 3 Spelling Corrector

About
=====
Updated for Python 3, based on Peter Norvig's
2007 version: http://norvig.com/spell-correct.html

Examples
========
.. code-block:: python

    >>> from autocorrect import spell
    >>> spell('TGhe')
    'The'

Installation
============
.. code-block:: bash

    pip3 install autocorrect
